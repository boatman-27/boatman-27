<h4 align="left">My first full-stack project was my bachelor's thesis, where I designed and developed a web platform to support the various parties involved in the seafood supply chain. The frontend was built using React.js, providing a user-friendly interface, while the backend was implemented in Solidity to create smart contracts on the blockchain. These contracts handled user transactions, licensing (e.g., granting access only to verified fishermen), and the addition of seafood items to the blockchain for secure and transparent tracking.</h4>
