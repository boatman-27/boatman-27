Aspiring Software Developer
---------------------------

My first full stack project was for my bachelor thesis were i made a designed a website that acts a platform for the different parties in the seafood supply chain. The frontend was designed using ReactJs and the backend was coded in solidity to create smart contracts that can handle transactions between users, licensing to allow users with a fisherman license and lastly a contract that handles adding seafood items to the blockchain.
